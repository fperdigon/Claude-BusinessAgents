## Goals

Upgrade the carousel output from plain text + Heroicons to visually rich, brand-specific carousels that include:

1. **10 background patterns** per brand â€” derived from the company's actual business identity, not generic tech aesthetics
2. **10 infographic layout types** â€” agent picks the best fit per slide based on content type
3. **Brand ownership** â€” backgrounds and infographics are generated by the brand skill and stored in the brand kit; the marketing skill reads and uses them
4. **Interactive visual approval** â€” backgrounds and infographics are previewed in a browser during brand skill execution so the founder can approve, reject, and iterate before anything is saved
5. **Multi-platform format support** â€” 8 formats across 4 groups (Square, Portrait, Landscape, Document); the marketing skill asks which platform/format at the start of each session and sizes the output accordingly

## Platform Formats

Eight formats are supported across four groups. The marketing skill presents this list and asks the founder which platform they are creating for at the start of each session.

| Slug | Dimensions | Ratio | Platform | Use |
|---|---|---|---|---|
| `linkedin-carousel` | 1080 Ã— 1080 px | 1:1 | LinkedIn | PDF carousel / document post |
| `instagram-square` | 1080 Ã— 1080 px | 1:1 | Instagram | Feed post, carousel |
| `instagram-portrait` | 1080 Ã— 1350 px | 4:5 | Instagram | Feed post (max portrait fill) |
| `stories` | 1080 Ã— 1920 px | 9:16 | Instagram / LinkedIn / Facebook | Full-screen tap-through story |
| `pinterest` | 1000 Ã— 1500 px | 2:3 | Pinterest | Standard pin |
| `presentation` | 1920 Ã— 1080 px | 16:9 | Google Slides / Keynote / PowerPoint | Widescreen slide deck |
| `link-preview` | 1200 Ã— 628 px | 1.91:1 | LinkedIn / Twitter/X / Facebook | Shared-link thumbnail |
| `a4-letter` | 794 Ã— 1123 px | A4/Letter | PDF / print | One-pager, leave-behind |

**Background scaling:** All SVG backgrounds use `viewBox` + `preserveAspectRatio="xMidYMid slice"`, so a single SVG file works at every format without modification.

**Infographic layout adaptations for extreme ratios:**
- `stories` (9:16): prefer vertically-stacked layouts â€” process steps, stats grid, quote box. Avoid comparison tables and hub-spoke which need horizontal space. Stack icon grid 2Ã—3 instead of 3Ã—2.
- `presentation` (16:9): prefer horizontally-arranged layouts â€” before/after split, comparison table, hub-spoke, icon grid 3Ã—2. Stats grid and funnel adapt well. Process steps should be horizontal rather than vertical.
- All other formats (1:1, 4:5, 2:3, A4): use layouts as designed with no adaptation needed.

### Quote / testimonial layout
Only used when an actual quote exists in interview reports (`outputs/ideas/<slug>/interview-insights-*.md`) or the founder provides one explicitly during the carousel session. Never fabricated. **Brand skill â€” visual theme phase:**
- Never save any background or infographic without founder approval from the interactive preview round

- Progress bar metrics must always be stated positively â€” invert the statement if needed

**Marketing skill â€” format, background + infographic:**
- Ask the founder which platform/format before asking for a topic â€” format determines card dimensions
- Size `.card` elements to the exact pixel dimensions of the chosen format slug
- Apply infographic layout adaptations for `stories` (9:16) and `presentation` (16:9) as described in Platform Formats

- Always read `memory/startup-context.md` to extract company name, city/region, industry, niche, and technology â€” extend keyword categories with these terms before background matching
