How to export to LinkedIn

  1. Open File → Print (or Ctrl+P / Cmd+P) in your browser
  2. Set destination to Save as PDF
  3. Under "More settings", set paper size to Custom and enter {{card-w}} × {{card-h}}
  4. Disable headers and footers
  5. Save the PDF — each slide becomes one page
  6. On LinkedIn, start a new post → click the document icon → upload the PDF
  7. LinkedIn displays each page as one carousel slide — add your post caption and publish
